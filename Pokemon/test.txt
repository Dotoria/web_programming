사람들은 왜 성격 테스트를 할까? - 홀웬 니콜라스 (북드림, 2022)

1. MBTI

외향성 - 내향성 : 에너지를 얻거나 집중하고자 하는 것에 대한 선호
감각 - 직관 : 수집하거나 믿는 정보의 종류에 대한 선호
사고 - 감정 : 의사 결정에 대한 선호
판단 - 인식 : 주변 세계와의 상호작용에 대한 선호

2. Insights Discovery

빨강 = 지휘자 : 결단력, 자립적, 용감
주황 = 동기부여자 : 적극적, 활발, 열정적
노랑 = 격려자 : 사교적, 낙천적, 자기 표출적
라임그린 = 조력자 : 사람을 끌고, 힘을 돋워주며, 이해심
초록 = 지원자 : 배려심, 협조적, 참을성
청록 = 조정자 : 사려 깊고, 외교적, 신뢰할 수 있는
파랑 = 관찰자 : 일관적, 신중, 체계
보라 = 개혁자 : 자기 관리, 헌신적, 실용적

3. Enneagram

1 = 개혁가 : 원칙적, 이상주의적
2 = 조력자 : 배려심, 대인 관계
3 = 성취자 : 적응력, 성공지향적
4 = 개인주의자 : 몽상적, 내성적
5 = 연구원 : 진지, 지적
6 = 충신 : 헌신적, 안전지향적
7 = 열렬한 지지자 : 바쁘고, 생산적
8 = 도전자 : 영향력, 지배적
9 = 중재자 : 여유, 자기를 내세우지 않음

4. Process Communication Mode(l?), PCM

1 = 사상가 : 활동 조직, 계획적
2 = 고집쟁이 : 자신의 가치관과 신념이라는 필터
3 = 조화론자 : 감정이라는 필터, 따뜻하고 온정적
4 = 반항아 : 해결책을 찾는, 창의적
5 = 몽상가 : 내성적, 상상력
6 = 주창자 : 매력적, 호감가는

5. Neuro~Linguistic Programming, NLP

시각 (Visual, V) : 보는 것을 선호
청각 (Auditory, A) : 듣는 것
촉각 (Kinesthetic, K) : 만지고 느끼는 것
청각 디지털 (Auditory Digital, AD) : 표상을 자극하지 않는 논리와 데이터

6. The Eysenck Personality Inventory, EPI : 과민성

외향적 : 사회적, 태평한, 낙천적
내성적 : 조용, 자기 성찰적, 내성적
신경(과민)증 : 정서적 고통과 불안정 일으키기 쉬움
안정적 : 차분, 정서적으로 안정적

7. The Hexaco Model of Personality Structure Inventory

정직-겸손 (Honesty-Humility, H) : 진실한, 정직한, 충실한, 겸손한 - 교활한, 부정직한, 탐욕스런, 가식적인, 뽐내는, 거만한
정서성 (Emotionality, E) : 감정적인, 과민한, 감상적인, 걱정하는, 연약한 - 용감한, 냉정한, 독립적인, 자신감 있는, 안정적인
외향성 (Extraversion, X) : 외향적, 적극적, 활발한, 사교적, 수다스러운, 명량한 - 수줍음이 많은, 수동적, 내성적, 속을 드러내지 않는
우호성 (Agreeableness, A) : 인내심이 많은, 아량 있는, 평화적인, 관대한, 온화한 - 성미가 까다로운, 싸움을 좋아하는, 완고한, 성마른
성실성 (Conscientiousness, C) : 체계적, 자제력이 있는, 부지런한, 배려심이 깊은, 꼼꼼한, 정확한 - 엉성한, 태만한, 부주의한, 게으른, 무책임한
개방성 (Openness of Experience, O) : 지적, 창의적, 비전형적, 혁신적, 역설적인 - 피상적인, 상상력이 부족한, 전형적인

8. The Birkman Method

행동형(빨강) : 행동 지향적, 지시와 건설, 단호한, 정력적인
분석형(노랑) : 분석적, 사려 깊은, 세부 지향적
소통형(초록) : 사람에 초점을 맞춤, 설득, 적극적, 경쟁심이 심한
사고형(파랑) : 지적인, 혁신과 설계, 통찰력이 있는

9. Hogan Personality Inventory, HPI

1차 척도
적응 : 자신감, 자존감, 침착함
야심 : 진취성, 경쟁심, 리더십에 대한 욕망
사회성 : 외향성, 사교성, 사회적 상호 작용에 대한 니즈
대인 민감성 : 눈치, 예민함, 관계 유지 능력
신중 : 자기 수양, 책임성, 성실성
탐구심 : 상상력, 호기심, 창의성
학습 접근 : 성과지향적, 비즈니스 및 기술 문제의 최신 정보

2차 척도
서비스 지향 : 배려심, 상냥한, 예의 바른
스트레스 내성 : 스트레스를 다룰 수 있음, 침착한
신뢰성 : 정직한, 성실한, 긍정적인
사무직 잠재력 : 지시를 따르는 능력, 세부적인 것, 명확한 소통
영업직 잠재력 : 에너지, 사교술, 문제 해결 능력
관리직 잠재력 : 리더십 능력, 기획, 의사 결정 기술

14. The Five Factor Model of Personality, Big 5

개방성 (Openness, O) : 상상력, 통찰력, 관심사 다양, 호기심, 새로운 것을 배우기, 새로운 경험
성실성 (Conscientiousness, C) : 신중함, 강한 절제력, 목표 지향적 행동, 체계적, 세부적인
외향성 (Extraversion, E) : 자극에 민감, 사교적, 수다스러운, 적극적, 감정 표현 풍부, 외향적이고 사교적인 분위기에서 에너지 얻기, 사람과 함께할 때 활기와 흥분
친화성 (Agreeableness, A) : 신뢰, 이타주의, 친절, 애정, 협조적, 공감 능력
신경과민성 (Neutoricism, N) : 침울함, 까다로움, 정서 불안, 감정 기복, 불안, 짜증, 우울감 경험

15. Carl Jung

외향적 사고형 : 합리적인 사고와 논리, 자신의 세계관이 옳은 것, 질서와 사실 선호
내향적 사고형 : 사실(외적 세계)보다 사상(내적 세계), 의문 제기, 이론 세우기
외향적 감정형 : 관습적, 개인적인 성공, 사람 다루기, 또래 집단과 어울리기
내향적 감정형 : 냉담, 내향적, 평화와 고요, 속마음 드러내지 않음
외향적 감각형 : 외부 세계의 대상과 감각, 세상을 있는 그대로, 인생 즐기기
내향적 감각형 : 경험되는 감각, 자신을 표현하기 어려움, 현실과 접촉 원치 않음
외향적 직관형 : 쉽게 지루, 새로운 아이디어 탐구, 결정이나 판단할 때 직관에 의존
내향적 직관형 : 꿈, 비전, 집단 무의식에 의한 신비로운 세계에 초점, 내면의 백일몽과 환상
