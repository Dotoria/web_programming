# 타입별 문항 설계 (108문항 개정판)

워크플로우: ① 타입별 표면/내면 성향 정의(도감·생김새·특성 기반) → ② 타입 상성을 성격 궁합으로 치환 → ③ 일상 행동 중심의 짧은 문항 작성 → ④ 복합 문항은 실제 그 타입 조합을 가진 포켓몬의 서사를 반영

## 개정 이력 (2026-07-14, 피드백 반영)

1. **문법·표현 폴리싱**: "정작"의 오용(구 49번) 수정, "자존심이 있어서라도"(구 65번) 삭제, 쉼표 앞뒤 연관성이 없던 복합 문항(구 43번 에스퍼|페어리)은 에스퍼 단일·페어리 단일 문항으로 분리. 전 문항을 더 짧고 명확하게 재작성.
2. **문항 수 77 → 108개**: 단일 5문항 × 18타입 = 90 + 복합 18문항.
3. **타입 간 완전 균등**: 복합 18문항은 18개 타입이 정확히 2번씩 등장하는 순환 구조 → **모든 타입이 정확히 7문항(단일 5 + 복합 2)에서 점수를 받는다.** 정규화(평균) 시 완전히 공평.
4. **특성(어빌리티) 반영**: 노말=적응력, 불꽃=맹화, 바위=옹골참, 풀=초식, 물=급류 등의 뉘앙스를 문항·설명에 녹임.
5. **채점 규칙 확장** (`type.js`):
   - 전 타입 평균의 최고-최저 차이가 0.35 이하(모두 가운데로 답한 경우 포함) → **노말 단일** ("균형" 그 자체가 노말의 정체성)
   - 1위 평균이 2위보다 0.75 이상 높음 → **단일 타입**
   - 그 외 → 상위 2개 **복합 타입**

## 최종 문항 구성 (108문항, `question.csv` 기준)

- **단일 타입 문항** 90개: 타입당 5개 × 18타입 (각 타입에 갈등 대응 문항 최소 1개 포함)
- **복합 타입 문항** 18개: CSV에 `타입1|타입2`로 기록, 하나의 답변이 두 타입에 동시에 점수를 준다
- **시간 예산**: 문항당 평균 8초 기준 108문항 ≈ 15분
- **타입 노출 방지**: 연속한 두 문항이 같은 타입을 공유하지 않고, 복합 문항이 연달아 나오지 않도록 스크립트로 배치·검증 (`scratchpad/build_questions.py` 시드 20260714)

### 복합 문항 18개 (18개 타입이 정확히 2번씩 등장하는 순환)

| 타입 조합 | 해당 조합 포켓몬 | 문항 |
|---|---|---|
| normal\|fairy | 푸크린 | 튀는 걸 좋아하진 않지만, 주변 사람들 기분을 살뜰히 챙기는 건 좋아한다. |
| normal\|psychic | 키링키 | 평소엔 무던하다는 말을 듣지만, 가끔 남들이 놓친 걸 혼자 알아챌 때가 있다. |
| grass\|psychic | 나시 | 내 리듬대로 느긋하게 움직이는데, 중요한 순간엔 감이 딱 맞아떨어질 때가 많다. |
| grass\|poison | 이상해꽃 | 먼저 부딪히는 일은 없지만, 누가 선을 넘으면 반드시 그만큼 돌려준다. |
| poison\|electric | 스트린더 | 신나면 누구보다 화끈하게 놀지만, 거슬리는 상대에겐 톡 쏘는 말이 먼저 나간다. |
| electric\|steel | 자포코일 | 판단이 필요한 순간엔 빠르게 결론을 내리고, 그 판단은 좀처럼 틀리지 않는다. |
| steel\|fighting | 루카리오 | 한번 세운 원칙은 손해를 보더라도 행동으로 지켜야 직성이 풀린다. |
| fighting\|fire | 번치코 | 힘에 부치는 일일수록 오히려 오기가 생겨서 끝을 보고야 만다. |

| ghost\|dragon | 기라티나 | 분위기가 너무 진지해지면 농담으로 넘기거나 슬쩍 자리를 피하고 싶어진다. |
| dragon\|ground | 한카리아스 | 어려운 일일수록 해보고 싶어지지만, 그 와중에도 손익 계산은 놓치지 않는다. |
| ground\|rock | 코뿌리 | 한번 자리를 잡으면 어지간한 자극에는 흔들리지 않고 묵묵히 그 자리를 지킨다. |
| rock\|dark | 마기라스 | 오래 지켜온 것에 자부심이 있어서, 남들이 뭐라 하든 크게 흔들리지 않는다. |
| dark\|ice | 포푸니라 | 평소엔 무심해 보여도, 필요한 순간엔 누구보다 빠르고 정확하게 움직인다. |
| ice\|water | 라프라스 | 힘들어 보이는 사람은 먼저 챙기면서도, 정작 내 얘기를 꺼내는 데는 시간이 걸린다. |
| water\|flying | 갸라도스 | 웬만하면 맞춰주지만, 갑갑한 상황이 이어지면 어느 순간 확 뒤집고 떠나고 싶어진다. |

> 구판의 복합 문항 중 순환 구조에 들어가지 못한 조합(fire\|flying, psychic\|fairy 등)은 삭제하거나 단일 문항으로 분해했다. 구판에서 fire\|fighting의 대표로 잘못 적혀 있던 블레이범(실제로는 불꽃 단일)은 번치코로 교체.

> ⚠️ 결과 화면의 "타입이 완벽하게 일치하는 포켓몬"은 `type.js`의 `PURE_POKEMON`(단일)·`COMBO_POKEMON`(복합, 지역 폼 제외) 표를 사용한다. 존재하지 않는 조합은 "도감에 없는 특별한 조합" 메시지로 대체.

## 타입별 단일 문항 (각 5개, ⚔=갈등 대응 문항)

### 노말 (Normal) — 표면: 무난함, 균형 / 내면: 어디에도 안 치우치는 균형감각
1. 여럿이 무언가를 정할 때, 굳이 내 의견을 내세우지 않아도 자연스럽게 어울린다.
2. 특기가 뭐냐고 물으면 딱히 떠오르지 않지만, 시켜보면 웬만한 건 다 해내는 편이다.
3. ⚔ 웬만한 일에는 먼저 한발 물러서지만, 아끼는 사람의 일이라면 얘기가 달라진다.
4. 어떤 모임에 가도 크게 튀지도, 겉돌지도 않는 편이다.
5. 낯선 환경에 놓여도 금방 적응해서 원래 있던 사람처럼 지낸다. (특성: 적응력)

### 불꽃 (Fire) — 표면: 열정, 다혈질 / 내면: 몰입해야 존재감을 느낌
1. ⚔ 부딪힐 일이 생기면 피하지 않고 정면으로 맞서서 결판을 내야 마음이 풀린다.
2. 새로운 목표가 생기면 재고 따지기 전에 일단 부딪히면서 방법을 찾는다.
3. 한번 욱하면 잘 가라앉지 않아서, 지나고 보면 좀 과했다 싶을 때가 있다.
4. 궁지에 몰릴수록 오히려 힘이 나서, 마지막에 뒷심을 발휘하는 편이다. (특성: 맹화)
5. 하고 싶은 일이 생기면 참고 기다리는 게 세상에서 제일 어렵다.

### 물 (Water) — 표면: 유연함, 포용력 / 내면: 강물처럼 끌어안고 바다처럼 품지만, 눌러 담다 한번에 터짐
1. 웬만한 일은 참고 넘기지만, 한계가 오면 스스로도 놀랄 만큼 크게 터진다. (특성: 급류)
2. ⚔ 처음 겪는 상황에서는 바로 뛰어들기보다 분위기부터 살핀다.
3. 딱히 해결책을 주지 않아도, 옆에 있어주는 것만으로 상대가 편해한다는 말을 듣는다.
4. 나와 많이 다른 사람도 일단 품고 보는 편이라, 주변에 다양한 사람이 모인다.
5. 어떤 사람을 만나도 그 사람에게 맞춰주는 게 그리 어렵지 않다.

### 풀 (Grass) — 표면: 평화, 느긋함 / 내면: 자기 속도를 지키고 싶은 것
1. 티 나게 나서기보다, 내 방식대로 조용히 하다 보면 어느새 결과가 따라온다.
2. 내 공간만큼은 늘 정돈된 상태로 두려고 한다.
3. 다들 당연하게 여기는 것도 종종 다른 각도에서 다시 생각해본다.
4. ⚔ 누가 부추기거나 도발해도 잘 넘어가지 않고 내 페이스를 지킨다. (특성: 초식)
5. 남들이 재촉해도, 결과물은 충분히 익을 때까지 기다렸다가 내놓는 편이다.

### 전기 (Electric) — 표면: 발랄, 즉흥 / 내면: 멈추면 오히려 불안
1. ⚔ 의견이 부딪히면 오래 끌지 않고 그 자리에서 확실히 매듭짓는다.
2. 기분이 좋으면 숨기지 못해서, 주변 분위기까지 덩달아 들뜬다.
3. 재미를 느끼기 시작하면 브레이크가 안 걸릴 만큼 몰입한다.
4. 좋은 생각이 떠오르면 바로 해봐야지, 묵혀두면 몸이 근질거린다.
5. 일정이 빈틈없이 차 있을 때 오히려 컨디션이 좋다.

### 얼음 (Ice) — 표면: 차가움, 거리감 / 내면: 미리 방어막을 치는 자기 보호
1. 무슨 생각 하냐는 질문을 받을 만큼 속마음이 표정에 잘 드러나지 않는다.
2. 누가 훅 다가오면 반가움보다 경계심이 먼저 든다.
3. ⚔ 마음이 상하면 맞서기보다 조용히 마음의 문을 닫아버린다.
4. 감정이 격해질수록 오히려 머리는 차갑게 식는 편이다.
5. 서로 적당한 거리를 지켜주는 관계가 가장 편하다.

### 격투 (Fighting) — 표면: 우직함, 근성 / 내면: 행동으로 증명해야 인정받는다는 믿음
1. 고민만 하는 것보다 몸을 움직여야 스트레스가 풀린다.
2. ⚔ 부당한 상황을 보면 그냥 지나치지 못하고 나서서 짚는다.
3. 긴 설명을 듣지 않아도, 상대의 태도와 기세만 보면 상황이 대충 파악된다.
4. 잔재주 없이 실력으로 겨루는 정정당당한 승부를 좋아한다.
5. 말로 하는 약속보다 행동으로 보여주는 사람을 믿는다.

### 독 (Poison) — 표면: 예민함, 톡 쏘는 한마디 / 내면: 먼저 다치기 싫어 무기부터 숨겨두는 방어기제
1. 새로 알게 된 사람은 겪어보기 전까지 온전히 믿지 않는다.
2. ⚔ 다툼이 예상되면 미리 대비해두고, 상대가 움직일 때까지 기다린다.
3. 대놓고 나서기보다 조용히 준비해서 한 번에 보여주는 쪽을 선호한다.
4. 나도 모르게 뼈 있는 한마디를 얹었다가, 상대가 움찔하는 걸 볼 때가 있다.
5. 남들이 무심코 하는 말이나 행동에도 신경이 곤두설 때가 많다.

### 땅 (Ground) — 표면: 든든함, 현실감각 / 내면: 단단하기보다 든든, 부드럽지만 크게 영향
1. 그럴듯한 계획보다 당장 실현 가능한지부터 따진다.
2. 힘들었던 시기를 지나온 덕분에, 지금은 웬만한 일에 잘 흔들리지 않는다.
3. ⚔ 다툼이 커질 것 같으면 감정보다 실리를 따져 상황부터 진정시킨다.
4. 주변 사람들이 힘들 때 기댈 사람으로 나를 먼저 떠올리는 편이다.
5. 강하게 밀어붙이지 않아도, 어느새 주변이 내 뜻대로 움직여 있을 때가 많다.

### 비행 (Flying) — 표면: 자유로움 / 내면: 회피성향이지만 자기만의 신뢰는 지킴
1. 가끔 아무에게도 말하지 않고 훌쩍 떠나고 싶은 충동이 든다.
2. ⚔ 껄끄러운 다툼이 생기면 맞서기보다 일단 그 자리를 벗어난다.
3. 얽매이는 건 싫지만, 한번 맡은 일과 약속은 끝까지 책임진다.
4. 한곳에 오래 머물면 답답해져서 자꾸 새로운 곳을 찾게 된다.
5. 일이 복잡하게 꼬일수록 한발 떨어져서 전체 그림부터 본다.

### 에스퍼 (Psychic) — 표면: 직관, 사색 / 내면: 남의 마음은 잘 읽지만 자기 감정은 모름
1. 설명하긴 어렵지만, 촉으로 내린 판단이 좋은 결과로 이어진 경험이 꽤 있다.
2. ⚔ 다툼이 날 낌새가 보이면 말이 오가기 전에 분위기를 풀어두는 편이다.
3. 가까운 사람에게 무슨 일이 생기면, 이유 없이 먼저 이상한 느낌이 들 때가 있다.
4. 말하지 않아도 상대의 기분 변화를 먼저 알아챈다. (구 43번 앞부분 분리)
5. 몸을 움직이는 것보다 생각에 잠겨 있는 시간이 더 많다.

### 벌레 (Bug) — 표면: 성실, 디테일 / 내면: 완벽하지 않으면 불안, 탈피하듯 성장
1. 남들은 그냥 넘어가는 작은 부분도 눈에 밟혀서 결국 손을 보고야 만다.
2. ⚔ 갈등이 생기면 어중간하게 덮지 못하고 끝을 볼 때까지 파고든다.
3. 지금은 부족해도 차근차근 쌓다 보면 결국 달라질 거라고 믿는다.
4. 거창한 목표보다 오늘 할 몫을 채우는 데서 만족을 느낀다.
5. 돌아보면 나는 완전히 다른 사람이 됐다 싶을 만큼, 몇 번씩 크게 변해왔다. (탈피)

### 바위 (Rock) — 표면: 단단함, 뚝심 / 내면: 단단하지만 부서질 수 있음, 날카롭게 찌르고 둥글게 구름
1. 한번 익숙해진 방식은 특별한 이유가 없는 한 바꾸지 않는다.
2. ⚔ 먼저 맞서지는 않지만, 옳다고 믿는 입장만큼은 절대 굽히지 않는다.
3. 손에 익은 물건은 낡아도 쉽게 버리지 못한다.
4. 아무리 큰 충격을 받아도 한 번에 무너지지는 않는 편이다. (특성: 옹골참)
5. 빙 둘러 말하기보다 핵심을 곧장 짚는 편이다. (날카로움)

### 고스트 (Ghost) — 표면: 은둔, 장난기 / 내면: 상처를 유머로 포장
1. 진짜 속마음은 아무리 친한 사이라도 전부 보여주지 않는다.
2. 북적이는 자리보다 혼자 보내는 시간이 훨씬 편하다.
3. 사람을 살짝 놀라게 하는 장난을 즐긴다.
4. ⚔(회피형) 모임에서 있는 듯 없는 듯 지내다가, 아무도 모르게 슬쩍 빠져나온 적이 있다.
5. 밝고 신나는 이야기보다, 어딘가 쓸쓸하거나 오싹한 이야기에 더 끌린다.

### 드래곤 (Dragon) — 표면: 강함, 자부심 / 내면: 약한 모습을 숨기지만 다정함을 들키고 싶음
1. 남들은 나를 강하고 자신감 넘치게 보지만, 실제로는 정 많고 여린 구석이 있다.
2. 다들 어렵다고 피하는 일일수록 승부욕이 생긴다.
3. ⚔ 내 사람이나 내 몫이 걸린 일에서는 먼저 물러서지 않는다. (구 65번에서 "자존심이 있어서라도" 삭제)
4. 어중간한 성과에는 만족하지 못한다. 기왕 하는 거라면 최고가 되고 싶다.
5. 칭찬도 좋지만 "역시 너밖에 없다"는 말을 들을 때 가장 뿌듯하다.

### 악 (Dark) — 표면: 날렵함, 자기기준 / 내면: 어둠이 편하고 허를 찌르는 데 능함, 미리 기대를 접는 방어기제
1. 남들이 정한 규칙보다 스스로 납득한 기준을 따르는 게 편하다.
2. ⚔ 다툼이 생기면 감정적으로 맞서기보다 차갑게 선을 긋고 거리를 둔다.
3. 믿기까지는 오래 걸려도, 한번 믿은 사람에게는 전혀 다른 모습을 보여준다.
4. 정면 대결보다는 상대의 허를 찌르는 쪽이 내 방식이다.
5. 환한 낮보다 조용한 밤이 편하고, 밤에 머리가 더 잘 돌아간다.

### 강철 (Steel) — 표면: 단단함, 원칙 / 내면: 담금질로 만들어진 단단함
1. 일할 때는 그날 기분보다 미리 정해둔 기준을 우선한다.
2. ⚔ 정말 중요한 부분에서는 다툼이 생기더라도 타협하지 않는다.
3. 맞다고 확신하면 주변에서 뭐라 해도 끝까지 밀고 나간다.
4. 감정에 호소하는 말보다 근거가 분명한 말에 움직인다.
5. 나만의 루틴이 무너지면 그날 하루가 통째로 흔들리는 기분이 든다.

### 페어리 (Fairy) — 표면: 사랑스러움 / 내면: 남을 챙기느라 자신을 못 챙김
1. 작고 여린 존재를 보면 괜히 마음이 쓰여서 더 챙겨주고 싶다.
2. ⚔ 분위기가 서먹해지면 먼저 다가가서 풀어보려 한다.
3. 누가 힘들어하면 내 일처럼 마음이 쓰여서 어떻게든 돕고 싶어진다.
4. 아끼는 사람을 위해서라면 손해나 위험도 기꺼이 감수한다. (구 43번 뒷부분 분리)
5. 선물이나 깜짝 이벤트처럼, 주변을 기분 좋게 만드는 일을 즐겨 챙긴다.
